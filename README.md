# 랜덤 타워디펜스 (Random Tower Defense)

스타크래프트2 아케이드 인기 유즈맵 **「랜덤 타워 디펜스」** 의 구조를 분석해
**Vue 3 + TypeScript(화살표 함수)** 로 재구성한 가로(横) 레이아웃 타워디펜스 게임.

- 게임 구조 분석: [ANALYSIS.md](ANALYSIS.md) (나무위키 기준 정본 설계 문서)
- 기술: Vue 3 `<script setup>` + TypeScript + Vite, 게임 필드는 Canvas 2D, HUD는 Vue 반응형.

## 실행

```bash
npm install
npm run dev      # http://localhost:5173 에서 플레이
```

> 타입체크: `npx vue-tsc --noEmit` (통과)

### ⚠️ 프로덕션 빌드(`npm run build`) 관련 주의
현재 이 PC 환경(**Windows + Node 24.13**)에서는 `vite build`(=rollup) 가 청크 렌더링
단계에서 네이티브 프로세스 크래시(`0xC0000409`, STATUS_STACK_BUFFER_OVERRUN)로 종료됩니다.
- **게임 코드와 무관**한 문제입니다. 10개 모듈짜리 hello-world 도 동일하게 크래시하고,
  타입체크는 정상 통과합니다. (rollup 네이티브/WASM 양쪽 모두 동일 증상)
- 흔한 원인: Node 24의 미성숙 + 백신/EDR 이 node 프로세스에 후킹하며 스택 쿠키를 깨뜨림.
- **해결책**: 빌드가 필요하면 **Node 20/22 LTS** 로 전환 후 `npm run build` 하세요.
  개발/플레이는 `npm run dev`(esbuild 기반)로 영향 없이 동작합니다.

## 게임 방법 (실제 RTD 구조)

0. 시작 시 **난이도 선택**(쉬움 300% / 보통 400% / 어려움 500% / 지옥 500%·목숨5).
1. **일반 타워(🎲, 100 미네랄)** 만 무작위로 나옵니다. 종족은 고르지 않습니다.
2. **같은 종류 타워 2개**가 모이면 타워를 선택해 **합성(⚙)** → 상위 등급으로 승급
   (일반→희귀→영웅→전설→**신**). 합성 결과의 종족·종류는 무작위.
3. 매 라운드 클리어 시 **+300 미네랄**. 판매는 **반값(+50)**.
4. **가스 도박(⛽)**: 미네랄 100 → 가스 20~120 무작위. 가스로 **3종족 동시 업그레이드**.
5. **보스(10·20·30·40·50 라운드)** 처치 시 **테라진(◆)**. 테라진은
   미네랄 400 / 가스 300 / **영웅 선택권**(◆1+100원) 으로 사용.
6. **개인 미션**([A]게 / [S]암흑광전사 / [D]대군주): 버튼으로 보스몹을 소환해 처치 시
   미네랄(+100/200/500). 쿨다운이 있고, 출구로 빠뜨리면 목숨 -1.
7. **퀘스트 15종**(일반/관문/희귀/영웅7·전체/전설6·전체/신5/3종족신/뫼비우스/라이프5/750킬/시간).
   전설 수집 보상으로 **전설 확정 선택권**(원하는 전설 무료 건설, 메뉴→테라진 탭).
8. **영웅~신 고유 스킬**(뉴 랜타디 23): 감속·기절·확률 다중타격·3단 충전·**식충 소환·폭탄 난사·
   듀크 대체모드·골리앗 이중무기**·약화/아군버프 등. 선택 타워 정보에 표시됩니다.
9. **50라운드 생존 = 승리**. 클리어 후 **무한모드**(테라진 222 + 51R~) 계속 가능.
   적이 출구로 빠져나가면 목숨 감소, 0이면 패배.
- 상단의 **⏸/1x/2x/3x** 로 일시정지·배속을 조절합니다.

### 종족 색깔
- **테란(T)**: 단일 고화력 — 보스에 강함 (불곰·궤멸충)
- **저그(Z)**: 광역 라인 클리어 — 잡몹에 강함 (울트라·가시지옥)
- **프로토스(P)**: 밸런스/광역 — 보스·라인 무난 (고위기사·사도)

## 폴더 구조

```
src/
  engine/
    types.ts        타입 정의
    balance.ts      밸런스 상수(라운드/경제/가스도박/테라진) — 난이도 튜닝 지점
    content.ts      타워 44종(뉴 랜타디 23 실측 DPS/스킬)·3종족·퀘스트
    roundHp.ts      라운드별 체력 50칸 실측 테이블
    path.ts         가로 맵 경로 + 위치 계산
    GameEngine.ts   건설·합성·가스도박·테라진·업그레이드·전투·퀘스트
  composables/
    useGame.ts      엔진 ↔ Vue 반응형 래퍼 + requestAnimationFrame 루프
  components/
    GameField.vue   Canvas 게임 필드(건설/합성 선택/영웅 배치)
  App.vue           가로 레이아웃 HUD(건설·3종족업글·테라진·인스펙터·퀘스트)
  main.ts, style.css
```

## 구현 범위
- **반영**: 일반-only 랜덤 건설, 합성 승급(5등급·등급별 판매가), 3종족 동시 업글, 가스 도박,
  테라진 3용도 + 전설 확정 선택권, 퀘스트 15종, 난이도 4종, 개인 미션 3단계(몹별 속도차),
  **영웅~신 고유 스킬 전부**(소환·폭탄·대체모드·이중무기·약화/버프 포함), 디버프/버프 시각표시,
  무한모드, 게임 클럭, 배속·일시정지, 보스 주기, 승패, 모바일 줌/팬.
- **보류**(ANALYSIS.md §7): 신 타워 자리 교체, 8인 경주 모드, 가운데 1개 제한 모드,
  난이도 5단계 세분화, 라운드별 몹 모델·이름.
