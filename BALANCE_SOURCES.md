# 밸런스 수치 출처 분류

각 수치가 **어디서 왔는지** 분류한 기록. 셋 중 하나:
- 🟢 **나무위키** — 나무위키 「랜덤 타워 디펜스(스타크래프트 2)」 문서에 명시된 값/사실
- 🔵 **사용자 확정** — 사용자가 직접 확인/지정해 준 값
- 🟡 **임의 설정** — 나무위키에 정보가 없어 개발(Claude)이 임의로 정한 값

> 코드 위치: 수치는 대부분 [src/engine/balance.ts](src/engine/balance.ts), 타워는 [src/engine/content.ts](src/engine/content.ts).
> 마지막 갱신: 로스터를 **뉴 랜타디 23 인게임 데이터(44종)** 로 전면 교체 + role/범위 분류 반영.
> ⚠️ 아래 🟢 「타워」 항목의 캐릭터/스킬 예시(울트라·수호군주·사도 분신 등)는 **나무위키 기준 클래식 정보**이며,
> 실제 구현 로스터는 뉴 랜타디 23(44종)이라 일부는 포함되지 않는다. 유닛 고유 스킬은 아직 미구현(평타+role+splash만).

## 🟢 나무위키 기반 (확인된 사실)

### 게임 구조
- 총 **50 라운드**, **10라운드마다 보스**, 50R 클리어 = 승리
- 무한모드: 50R 클리어 시 **테라진 222** 지급, 51R~ 계속

### 경제
- 라운드 클리어 **미네랄 +300**
- 일반 타워 건설 **100 미네랄**
- 판매 **50% 환급**
- **처치당 미네랄 없음** (라운드 보상만) → `mobBounty/bossBounty = 0`
- 미네랄 100 → **가스 도박 20~120, 10단위 랜덤**

### 업그레이드
- 3종족 구분, **최대 100업**
- **단리(linear) 계산**
- **단계별 추가 가스 +2** (2017 패치로 5→2)

### 테라진
- 보스마다 **+1**(총 5), 누적 **750킬 +2**, **3종족 신 +1** → 최소 6, 최대 7
- 사용: **미네랄 400** / **가스 300** / 영웅(미네랄 100 + 테라진 1)

### 타워
- **5등급**(일반·희귀·영웅·전설·신). 구현 로스터(뉴 랜타디 23): 일반8·희귀8·영웅8·전설8·**신12** = 44종
- **합성**: 같은 등급·같은 포탑 2개 → 상위 등급(무작위)
- **전설·신 타워의 DPS·사거리 = 나무위키 실측값**
  (예: 울트라 dps5000/사거리5.4, 궤멸충 dps5666/7.2, 우주모함 13.2, 수호군주 666/10.8 …)
- 일반·희귀·영웅 **사거리 서술 반영**: 광전사=근접 짧음(3), 공성전차=사거리만 김(13), 암흑기사=근접(3), 골리앗=사거리 김(11), 허크=짧음(5)
- 스킬 메커니즘: 사도 **분신(2%)**, 불곰 **감속**, 수호군주·악령 **3마리 타격**, 고위기사 **스플래시 광역**

### 개인 미션
- 게 **HP 3000**/광물 100, 암흑광전사 **HP 30000**/200, 대군주 **HP 175000**/500
- **5분(300초) 쿨다운**, 난이도에 따라 체력 변동(→ 기본값 × 난이도배율)
- 실패 시 목숨 **-1**(뉴버전)

### 퀘스트
- 일반 8종 +300, 영웅 7개 +300, 전설 6개 +300, 신 5개 +500, 3종족 신 +테라진1, 750킬 +테라진2
- 시간 경과 **7분 +200 / 12분 +300 / 18분 +테라진2**

### 난이도
- 체력 **300~500%**, **지옥(500% 목숨5)** = 시작 시 **미네랄 200·가스 100** 보너스

## 🟡 임의 설정 값 (나무위키에 정보 없음)

### 경제·시작
- 일반 난이도 목숨 **20** (지옥 목숨5는 나무위키)

### 난이도 곡선
- ~~잡몹 체력 공식~~ → **사용자 실측 테이블로 교체됨**(위 🔵 참고). `enemyHpBase/enemyHpGrowth/bossHpMultiplier`는 더 이상 사용 안 함(무한모드 외삽만 +15%/R 임의)
- 라운드당 잡몹 **20마리**(보스 1마리) — 🔵 사용자 확정
- 스폰 간격 **0.5초** (임의)

### 생명 손실
- 잡몹 관통 **-1**, 보스 관통 **-10** (미션 관통 -1은 나무위키)

### 타워 스탯 — 🔵 사용자 제공으로 전면 교체됨
- **로스터 전체 = 뉴 랜타디 23 인게임 데이터**(44종, DPS·공속·사거리·종족). `src/engine/content.ts` `DEFS`.
- (이전의 임의 DPS/공속/사거리는 폐기)

### 타워 공통 파생값
- **role(라인/보스/밸런스)·범위(AoE) 여부 = 🟢 나무위키 서술 기준** 분류
  (범위 20종 / 단일 24종, 보스특화 5종: 유령·토르·승천자·모조·모한다르)
- 🟡 **splash 반경 타워별 38~62px** — 유닛 특징 + 등급대비 DPS 고려해 임의 차등(고DPS/확률형↓, 저DPS/상시광역↑)
- 🟡 **보스 추가피해**: 라인 ×1.0 / 밸런스 ×1.4 / 보스 ×2.2 — 계수만 임의
- 🟡 캐릭터 아이콘(이모지) 전부
- **공격 횟수(hits) 반영됨**: 주기당 hits발 발사(여러 적에 분산), 발당=DPS×공속÷hits → DPS 유지

### 스킬 세부 파라미터 (메커니즘은 나무위키, 수치는 임의)
- 분신 최대 5스택, 감속 계수 0.5·지속 1.5초, 다단히트 3마리·반경 80px
- (스턴 0.8초 파라미터는 코드에 남아있으나 현재 사용하는 타워 없음)

### 렌더링·기타
- 맵 셀 크기 **48px**, 필드 576×576
- 미션몹 이동속도 `55 × 0.8`

## 변경 이력
- 나무위키 불일치 4건 수정:
  1. 사도 분신 확률 3% → **2%**
  2. 개인미션 쿨다운 45/60/75초 → **300초(5분)**
  3. 개인미션 체력 라운드비례 → **나무위키 절대값(3000/30000/175000) × 난이도**
  4. 고위기사 스킬 스턴 → **스플래시 광역**
